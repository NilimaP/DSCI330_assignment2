About the Program

Winona State University has created the first undergraduate program in Data Science in the upper Midwest. This degree program requires knowledge from statistics, computer science, and a domain-specific application area. This program was developed in consultation with several industry partners which ensures graduates will have the skills necessary to meet the rapidly growing demand for data scientists.
